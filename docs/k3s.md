# Kubernetes (k3s)

Active lightweight Kubernetes cluster
running inside cloud VMs (vm01, vm02, vm03).

Current Ingress:
- Traefik (Default)
- Homepage at `homepage.home`
- Grafana at `grafana.home`

Features:
- Immutable services
- Automated host updates & reboots
- Unified DNS via OpenWrt
- Persistent storage for workloads via NFS
- Full cluster monitoring
